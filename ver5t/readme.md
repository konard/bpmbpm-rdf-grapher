### TriG version
Каждая схема - это TriG, если в RDF-данных несколько TriG, то показывается только первый.

### run
https://bpmbpm.github.io/rdf-grapher/ver5t/
#### Новые возможности ver5t:
- Поддержка TriG формата с именованными графами
- Отображение rdfs:label именованного графа как заголовка диаграммы (вверху)
- Пример "Trig VAD" — все объекты Turtle VAD в одном именованном графе vad:Trig1

#### Пример использования:
Создайте TriG данные с именованным графом и его rdfs:label:
```
@prefix vad: <http://example.org/vad#> .
@prefix rdfs: <http://www.w3.org/2000/01/rdf-schema#> .

vad:Trig1 rdfs:label "Схема процесса TriG1" .

vad:Trig1 {
    # Содержимое графа...
}
```
Заголовок "Схема процесса TriG1" отобразится вверху диаграммы VAD.
