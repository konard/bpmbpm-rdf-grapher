# Модуль 7_info -- Prefixes

## Назначение

Модуль отображает список RDF префиксов пространств имен, извлеченных из загруженных данных. Предоставляет справочную информацию о используемых пространствах имен (например, `vad:` -> `http://example.org/vad#`, `rdf:` -> `http://www.w3.org/1999/02/22-rdf-syntax-ns#`). По умолчанию окно развернуто (настройка в `config.json`).

## Файлы модуля

| Файл | Назначение |
|---|---|
| `7_info_ui.js` | UI функции: генерация таблицы префиксов с пространствами имен |

## Ключевые функции

- **displayPrefixes(prefixes)** -- отображение таблицы префиксов в HTML
- **updatePrefixesDisplay()** -- обновление отображения при загрузке новых данных

## Зависимости от других модулей

- **9_vadlib** -- глобальная переменная `currentPrefixes` с текущими префиксами
- **2_triplestore** -- извлечение префиксов при парсинге RDF данных

## Расширение модуля

- **Редактирование префиксов:** добавьте возможность ручного редактирования/добавления префиксов
- **Экспорт:** добавьте кнопку экспорта списка префиксов в формате SPARQL PREFIX
