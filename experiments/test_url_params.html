URL Structure

https://bpmbpm.github.io/rdf-grapher/ver1/?rdf=<URL_ENCODED_RDF>&from=<FORMAT>&to=<OUTPUT>

Parameters:
- rdf: URL-encoded RDF data
- from: Input format (ttl, nt, nq, trig)
- to: Output format (svg, png)
