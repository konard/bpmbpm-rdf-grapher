Test Parent Selector Labels - Issue #414

Issue: #414

Problem: PR #413 fixed label display for root options (vad:ptree, vad:rtree), but other items in the vad:hasParentObj dropdown still showed only ID without label.

Root Cause: The funSPARQLvalues function strips OPTIONAL blocks from SPARQL queries, so rdfs:label is never returned in query results.

Solution: Added enrichResultsWithLabels() function that fetches rdfs:label from RDF store after SPARQL query execution.
