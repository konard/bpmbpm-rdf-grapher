# Модуль 6_legend -- Легенда стилей

## Назначение

Модуль отображает легенду используемых визуальных стилей для узлов и ребер графа. Помогает пользователю интерпретировать цветовую кодировку и форму узлов на VAD диаграмме. По умолчанию окно развернуто (настройка в `config.json`).

Легенда включает:
- Цвета и формы узлов для каждого типа (процесс, исполнитель, группа исполнителей, TriG)
- Стили узлов для подтипов процессов (Detailed, notDetailed, DetailedChild и др.)
- Стили ребер для различных предикатов (hasNext, includes, hasExecutor)

## Файлы модуля

| Файл | Назначение |
|---|---|
| `6_legend_ui.js` | UI функции: генерация HTML-таблицы с примерами стилей узлов и ребер |

## Ключевые функции

- **generateLegend()** -- генерация HTML-кода легенды на основе текущей конфигурации стилей
- **updateLegend()** -- обновление легенды при смене режима визуализации

## Зависимости от других модулей

- **9_vadlib** -- конфигурация стилей узлов и ребер, типы VAD
- **5_publisher** -- текущий режим визуализации для определения набора стилей

## Расширение модуля

- **Новые стили:** при добавлении новых типов узлов или ребер обновите генерацию легенды в `6_legend_ui.js`
- **Интерактивная легенда:** добавьте возможность фильтрации узлов по клику на элемент легенды
