# Модуль 4_resSPARQL -- Result in SPARQL

## Назначение

Модуль отображает результат работы модуля Smart Design в виде сформированных SPARQL запросов (INSERT DATA, DELETE DATA). Позволяет пользователю просмотреть сгенерированный запрос перед его применением, скопировать текст запроса и выполнить его для модификации текущих RDF данных.

## Файлы модуля

| Файл | Назначение |
|---|---|
| `4_resSPARQL_ui.js` | UI функции: текстовое поле для отображения SPARQL запроса, кнопки копирования и применения |

## Ключевые функции

- **displaySparqlResult(sparqlText)** -- отображение сформированного SPARQL запроса в текстовом поле
- **applySparqlResult()** -- применение отображенного SPARQL запроса к текущим RDF данным
- **copySparqlResult()** -- копирование текста запроса в буфер обмена

## Зависимости от других модулей

- **3_sd** -- получает сформированные SPARQL запросы от модуля Smart Design
- **2_triplestore** -- применяет SPARQL запросы к текущим RDF данным
- **9_vadlib** -- глобальные переменные и утилиты

## Расширение модуля

- **Подсветка синтаксиса:** добавьте CSS стили для SPARQL ключевых слов в отдельный файл `4_resSPARQL.css`
- **История запросов:** реализуйте хранение истории выполненных запросов в `4_resSPARQL_logic.js`
