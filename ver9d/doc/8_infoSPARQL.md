# Модуль 8_infoSPARQL -- SPARQL запросы

## Назначение

Модуль предоставляет интерфейс для ввода и выполнения произвольных SPARQL SELECT запросов к загруженным RDF данным. Использует собственный SPARQL движок `funSPARQLvalues` из модуля `9_vadlib`. Позволяет пользователю исследовать данные через SPARQL запросы без использования внешних SPARQL endpoint.

Предустановленный запрос по умолчанию:
```sparql
SELECT ?s ?p ?o
WHERE {
    ?s ?p ?o .
}
```

## Файлы модуля

| Файл | Назначение |
|---|---|
| `8_infoSPARQL_ui.js` | UI функции: текстовое поле SPARQL запроса, кнопка выполнения, область отображения результатов |
| `8_infoSPARQL_sparql.js` | Предустановленные SPARQL запросы и утилиты форматирования результатов |

## Ключевые функции

- **executeSparqlQuery()** -- выполнение SPARQL запроса из текстового поля через `funSPARQLvalues()`
- **displaySparqlResults(results)** -- отображение результатов запроса в табличном виде
- **loadPresetQuery(queryName)** -- загрузка предустановленного SPARQL запроса в текстовое поле

## Зависимости от других модулей

- **9_vadlib** -- `funSPARQLvalues()` для выполнения SPARQL запросов, `generateSparqlPrefixes()` для формирования префиксов
- **2_triplestore** -- текущие RDF данные (`currentQuads`, `currentPrefixes`)

## Расширение модуля

- **Предустановленные запросы:** добавьте новые запросы в `8_infoSPARQL_sparql.js`
- **Поддержка CONSTRUCT/ASK:** расширьте обработку типов запросов в `8_infoSPARQL_ui.js`
- **Автодополнение:** реализуйте подсказки для имен переменных и префиксов при вводе запроса
